# CLI 工具

fastapi-easy 提供命令行工具来简化开发。本指南介绍如何使用 CLI 工具。

---

## 初始化项目

```bash
fastapi-easy init --name my_api
```

---

## 生成代码

```bash
fastapi-easy generate --model Item --fields "name,price,description"
```

---

## 查看版本

```bash
fastapi-easy version
```

---

## 查看项目信息

```bash
fastapi-easy info
```

---

## 查看项目状态

```bash
fastapi-easy status
```
